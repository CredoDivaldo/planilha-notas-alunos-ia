# Story 1.1: Bootstrap da documentação inicial

## Status
Draft

## Story
**As a** equipa de desenvolvimento,
**I want** uma estrutura documental mínima do projecto (PRD, arquitectura e stories),
**so that** o ciclo AIOX de criação/validação/implementação possa arrancar sem bloqueios.

## Acceptance Criteria
1. Existe a pasta `docs/` no projecto.
2. Existem as pastas `docs/prd`, `docs/architecture` e `docs/stories`.
3. Existe `docs/prd/index.md` com escopo inicial do produto.
4. Existe `docs/architecture/index.md` com visão técnica inicial.
5. Existe uma story inicial em `docs/stories/` para iniciar o ciclo de desenvolvimento.
