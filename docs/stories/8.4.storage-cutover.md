### CLI subcommand pattern

Reuse the existing argparse subparsers pattern in `cli.py`. The subcommand:
1. Globs the given dir for `*_students*.csv` and `*_grades*.csv` (or any .csv)
2. For each, calls `parse_csv` from `legacy_import`
3. Validates using `validate_students_csv` / `validate_grades_csv`
4. Performs the same bulk insert as `ingest.py` (session.add + commit)
5. Emits a JSON report: `{"files_processed": N, "rows_imported": N, "rejected_rows": N, "errors": [...]}`
