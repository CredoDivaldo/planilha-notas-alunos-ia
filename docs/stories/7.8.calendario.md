## Story Statement

**As a** professor or student,  
**I want** a monthly calendar showing academic events (exams, tests, deadlines),  
**so that** professors can manage event schedules and students can see upcoming events relevant to their context without being able to edit them.

### Geração de `.ics`

O formato iCal básico:
```
BEGIN:VCALENDAR
VERSION:2.0
BEGIN:VEVENT
DTSTART:20260617T090000
DTEND:20260617T120000
SUMMARY:Exame Final Inglês
LOCATION:Sala 3
END:VEVENT
END:VCALENDAR
```
Usar `Blob` + `URL.createObjectURL` para download sem servidor.

### Acessibilidade

- EventDots: `<button type="button" aria-label="17 Junho - Exame Final Inglês Técnico">` — nunca apenas `<div>`
- Modal: focus trap, `aria-labelledby="event-modal-title"`, Escape fecha
- RadioGroup de tipos: `<fieldset><legend>Tipo de evento</legend>...` com cor também em texto (não só cor)
- Calendário: dias sem eventos têm `aria-label="5 de Junho, sem eventos"`
