### Design Tokens

Tokens definidos em dois locais para máxima compatibilidade:
- `src/index.css` — via `@theme {}` (Tailwind v4 CSS-first)
- `tailwind.config.ts` — via `theme.extend.colors` (compatibilidade)

### API Helper

`src/lib/api.ts` exporta:
```ts
export async function apiFetch<T>(path: string, options?: RequestInit): Promise<T>
```
- Prefixo automático com `import.meta.env.VITE_API_BASE_URL`
- Header `Authorization: Bearer {token}` quando token disponível no localStorage
- Lança erro com mensagem do backend em caso de status >= 400

### Acessibilidade
- ESLint configurado; shadcn/ui components excluídos do `react-refresh/only-export-components` via `globalIgnores` em `eslint.config.js`
