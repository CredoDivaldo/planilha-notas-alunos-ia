## Story Statement

**As a** professor,  
**I want** a page to create, view, edit, and delete academic contexts (class + subject + semester + shift combinations),  
**so that** I can manage the structural units that group my students and grades before starting any upload or publication workflow.

---

## Acceptance Criteria

- [x] **AC1:** A página `/contextos` renderiza com `AppHeader` (tab "Contextos" activo), título "Gestão de Contextos Académicos" e botão primário "+ Criar Contexto" no canto superior direito
- [x] **AC2:** Os contextos são carregados via `GET /academic-contexts/` ao montar a página; enquanto carrega, é exibido um spinner centrado
- [x] **AC3:** A tabela mostra colunas: Turma, Disciplina, Semestre, Turno, Alunos (contagem), Acções (ícones Editar + Eliminar); linhas são clicáveis para abrir o painel de detalhe
- [x] **AC4:** A barra de filtros tem: campo de pesquisa de texto livre (filtra por turma ou disciplina em tempo real no cliente), Select "Semestre" e Select "Turno"; os filtros combinam-se sem reload de página
- [x] **AC5:** Ao clicar numa linha da tabela, abre-se um painel de detalhe lateral (slide-in) ou abaixo da tabela com: sub-tabela de componentes de avaliação (Componente, Peso%, Acções), botão "+ Adicionar componente", contagem de estudantes + botões "Ver estudantes" e "+ Importar via CSV", e delegado atribuído com botão "Alterar delegado"
- [x] **AC6:** O modal "Criar Contexto" abre ao clicar "+ Criar Contexto"; tem campos: Turma (text, obrigatório), Disciplina (Select, obrigatório), Semestre (Select, obrigatório), Turno (Select, obrigatório), Curso (Select, opcional)
- [x] **AC7:** O modal "Criar Contexto" tem a secção de Componentes de Avaliação: tabela dinâmica com campos Componente (text) e Peso (%) por linha, botão "+ Componente" para adicionar linhas, botão 🗑 por linha para remover, totalização em tempo real — o botão "Criar Contexto" está desactivado se o total dos pesos for diferente de 100%
- [x] **AC8:** A validação de pesos mostra aviso `"⚠️ Os pesos devem somar 100%. Actual: X%"` em tempo real quando o total for diferente de 100%
- [x] **AC9:** Ao submeter "Criar Contexto" com dados válidos, chama `POST /academic-contexts/`; em caso de sucesso, fecha o modal, actualiza a tabela e mostra `StatusMessage` success; em caso de erro, mostra a mensagem de erro do backend no modal
- [x] **AC10:** O ícone Editar (✏️) na tabela abre o modal com os dados do contexto pré-preenchidos; ao submeter, chama o endpoint de update (PUT/PATCH `/academic-contexts/{id}`)
- [x] **AC11:** O ícone Eliminar (🗑) na tabela abre um `Dialog` de confirmação com texto "Eliminar o contexto [Turma · Disciplina]? Esta acção não pode ser revertida."; confirmar chama `DELETE /academic-contexts/{id}`; ao eliminar, a linha é removida da tabela
- [x] **AC12:** O botão "+ Importar via CSV" no painel de detalhe reutiliza o mesmo `FileDropzone` do Painel Professor (Step 1) para importar estudantes para o contexto seleccionado via `POST /students/upload`
- [x] **AC13:** A página é acessível WCAG AA: a tabela tem `<th scope="col">` em todos os cabeçalhos; o modal tem `role="dialog"`, `aria-labelledby`, e focus trap; Escape fecha o modal

### Endpoints API Relevantes

| Endpoint | Método | Uso |
|----------|--------|-----|
| `GET /academic-contexts/` | GET | Listar contextos do professor |
| `POST /academic-contexts/` | POST | Criar novo contexto |
| `PUT /academic-contexts/{id}` | PUT | Actualizar contexto |
| `DELETE /academic-contexts/{id}` | DELETE | Eliminar contexto |
| `POST /students/upload` | POST | Importar estudantes via CSV |

### Validação de Pesos

A validação de pesos deve ser reactiva:
```ts
const totalWeight = components.reduce((sum, c) => sum + Number(c.weight), 0)
const isValid = Math.abs(totalWeight - 100) < 0.01 // tolerância float
```
O botão "Criar/Guardar" usa `disabled={!isValid}`.

### Acessibilidade

- Tabela: `<table aria-label="Contextos Académicos">`, `<th scope="col">` em cada cabeçalho
- Modal: `role="dialog"`, `aria-labelledby="modal-title"`, focus trap implementado (primeiro elemento focusável ao abrir, Escape fecha)
- Botões de acção na linha (✏️, 🗑): `aria-label="Editar ING-T1"` e `aria-label="Eliminar ING-T1"` com o nome do contexto
